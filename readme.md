# webgl

Examples, tutorials, mini-projects, ...

## Installation

```
npm i -g webpack nodemon
npm install
```

## Usage

Build the client side and watch for changes:
```
webpack -w
```

Start the express server:
```
nodemon server/server.js -w server/
```
